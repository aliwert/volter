# Volter Guide

A production-grade, async-first web framework for Rust, built on [hyper],
[tokio], and [tower].

This guide complements the [API reference on docs.rs] with narrative,
task-oriented content: building your first API, adding extractors, composing
middleware, testing handlers, and more.

[hyper]: https://hyper.rs
[tokio]: https://tokio.rs
[tower]: https://docs.rs/tower
[API reference on docs.rs]: https://docs.rs/volter/latest/volter/
